# Server

Running the server:

    cd katseyl-server
    mvn spring-boot:run

The server will run on port 8080 by default.

# Client
Installing the prerequisites:

    # if missing angular cli (ng)
    npm install -g @angular/cli

Installing the client:

    cd katseyl-client
    npm install
    
Running the client:

    ng serve

The client will run on port 4200 by default.
